# Meem Browser Bridge Extension

Chrome Manifest V3 extension for Meem Browser Bridge.

It keeps a small popup UI, polls the local loopback bridge, executes browser commands in the user's existing Chrome session, and posts results back to the bridge.

## Load Unpacked

1. Open `chrome://extensions`.
2. Enable Developer Mode.
3. Click **Load unpacked**.
4. Select this `extension/` directory.
